# Location Tracker

A React Native app that tracks and displays your current location with search functionality.

## Setup

```bash
# Install dependencies
npm install

# Start the app
npm start
```

## Features
- Real-time location tracking
- Location search with geocoding
- Dark mode support
- GPS coordinates display
- Address lookup

## Tech Stack
- React Native
- Expo
- React Navigation
- React Native Maps
- Expo Location
